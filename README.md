# Тестовое задание для веб-студии Oxem

[посмотреть лендинг](https://oxem-test-task-nine.vercel.app/)


#### установка проекта
```
 npm install
```

#### запуск проекта
```
 npm run dev
```


#### В проекте использованы:
1. Next.js
2. TypeScript
3. Стейт менеджер Zustand
4. SCSS
5. GSAP
6. Swiper
7. React-hook-form + yup
